Settings

OSC UDP Port
The UDP port the HUD listens on for incoming OSC messages. Default: 52100. Range: 1024–65535.
OSC TCP Port
Reserved TCP port for future OSC-over-TCP support. Default: 52101.
HUD Width / Height
Dimensions of the HUD display window in pixels. Default: 800 × 300.
Default Text Color
The default color used for HUD text when no color is specified in an OSC message.
Default Background
The default background color of the HUD window.
Always on Top
When enabled, the HUD window stays above all other windows.
Auto-fit Font Size
Automatically scales text to fill the HUD window. When enabled, the fixed font size setting is hidden.
Fixed Font Size
A fixed font size in pixels, used when auto-fit is disabled. Default: 72. Range: 8–500.

OSC API

Send OSC messages over UDP to control the HUD. All addresses are prefixed with /sndwrks/hud.

/sndwrks/hud/message/single

Display a single message on the HUD.

ArgTypeDescription
1stringMessage text
2stringColor name or hex (optional)
/sndwrks/hud/message/single "Hello World"
/sndwrks/hud/message/single "Hello" "red"

/sndwrks/hud/message/lines

Display multiple lines. If the last argument is a valid color, it is used as the text color.

ArgTypeDescription
1…NstringMessage lines
laststringColor name or hex (optional — only used if it is a valid color)
/sndwrks/hud/message/lines "Line 1" "Line 2" "Line 3"
/sndwrks/hud/message/lines "Line 1" "Line 2" "blue"

/sndwrks/hud/message/flash

Flash a message for a given duration, then fade back to the default background.

ArgTypeDescription
1stringMessage text
2floatDuration in seconds
3stringColor name or hex (optional)
/sndwrks/hud/message/flash "STANDBY" 2.0
/sndwrks/hud/message/flash "GO" 1.5 "green"

/sndwrks/hud/clear

Clear the HUD display.

/sndwrks/hud/clear

/sndwrks/hud/color

Set the persistent text color (overrides the default until changed).

ArgTypeDescription
1stringColor name or hex
/sndwrks/hud/color "yellow"
/sndwrks/hud/color "#ff6b6b"

/sndwrks/hud/background

Set the background color.

ArgTypeDescription
1stringColor name or hex
/sndwrks/hud/background "#1a1a2e"

/sndwrks/hud/fontsize

Override the font size. Send 0 to return to auto-fit mode.

ArgTypeDescription
1intFont size in pixels (0 = auto)
/sndwrks/hud/fontsize 48
/sndwrks/hud/fontsize 0

Colors
